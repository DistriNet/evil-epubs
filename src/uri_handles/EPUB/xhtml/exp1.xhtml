URI handles

Not only can URIs include a reference to the requested resource, they can also refer to a requested application to open this resource. In this experiment, the EPUB will try to open various applications by leveraging specific URIs. Each part of the experiment will have to be started manually by clicking the associated button or link. Note that these actions can be automated through JavaScript, but are simply made manual for reasons of usability.
